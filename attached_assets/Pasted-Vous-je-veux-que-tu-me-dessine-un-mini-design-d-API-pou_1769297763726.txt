Vous

je veux que tu me dessine un mini design d’API pour MusicBrainz + Spotify,

Ulysse

OK, je te le dessine.

On part sur un mini design d’API qui met MusicBrainz derrière pour les métadonnées, Spotify pour la lecture, et qui expose une couche simple pour moi.

Je te fais ça en endpoints + JSON typique.

1️⃣ Namespace général
On met tout sous :

/api/music/*
Avec deux sous-blocs logiques :

/api/music/meta/* → métadonnées (MusicBrainz)
/api/music/player/* → contrôle de lecture (Spotify)
2️⃣ Métadonnées (MusicBrainz) – /api/music/meta/*
2.1 Recherche d’un morceau
GET /api/music/meta/track?artist=PNL&title=Au%20DD
Réponse JSON

{
  "artist": "PNL",
  "title": "Au DD",
  "album": "Deux Frères",
  "releaseYear": 2019,
  "durationMs": 268000,
  "musicbrainz": {
    "recordingId": "uuid-recording",
    "releaseId": "uuid-release",
    "artistId": "uuid-artist"
  },
  "genres": ["French hip hop", "Cloud rap"],
  "spotify": {
    "trackId": "spotify:track:6b1Cdm6FSx0jY3f3V40m2z",
    "previewUrl": "https://p.scdn.co/mp3-preview/...",
    "externalUrl": "https://open.spotify.com/track/..."
  }
}
Usage pour moi :

Quand tu dis “parle-moi de ce morceau”, je tape là-dessus, je peux te donner album, année, etc.
Je peux aussi enchaîner sur la lecture (via spotify.trackId).
2.2 Infos artiste

{

Usage pour moi :

Pour du contexte quand tu parles d’un artiste,
Pour proposer des albums à lancer sur Spotify.

3️⃣ Player (Spotify) – /api/music/player/*
Ici tu abstrais complètement Spotify, moi je ne parle qu’à toi.

3.1 Morceau en cours
GET /api/music/player/now-playing
Réponse JSON

{
  "isPlaying": true,
  "device": {
    "name": "Salon Echo",
    "type": "speaker",
    "volume": 72
  },
  "track": {
    "artist": "Damso",
    "title": "Macarena",
    "album": "Ipséité",
    "durationMs": 210000,
    "progressMs": 92000,
    "spotifyId": "spotify:track:xxx",
    "musicbrainzId": "uuid-recording",
    "imageUrl": "https://i.scdn.co/image/..."
  }
}
Usage pour moi :

Répondre à “c’est quoi ce que j’écoute là ?”
Proposer :
“tu veux que je te mette l’album depuis le début ?”
“on le remet depuis le début ?”
3.2 Lancer un morceau précis
POST /api/music/player/play-track
Content-Type: application/json

{
  "spotifyId": "spotify:track:6b1Cdm6FSx0jY3f3V40m2z"
}
Réponse

{
  "status": "ok"
}
Flux typique pour moi :

Tu dis :
“Mets Au DD de PNL”
Moi → /api/music/meta/track?artist=PNL&title=Au%20DD
Je récupère spotify.trackId
J’appelle /api/music/player/play-track avec ce trackId.
3.3 Jouer une playlist / album
POST /api/music/player/play-context
Content-Type: application/json

{
  "type": "album",
  "spotifyId": "spotify:album:xxx"
}
ou

{
  "type": "playlist",
  "spotifyId": "spotify:playlist:xxx"
}
Usage :

“Lance Ipséité depuis le début”
“Mets ma playlist chill”
3.4 Contrôles simples
POST /api/music/player/pause
POST /api/music/player/resume

POST /api/music/player/next
POST /api/music/player/previous

POST /api/music/player/volume
Content-Type: application/json

{
  "volume": 35
}
