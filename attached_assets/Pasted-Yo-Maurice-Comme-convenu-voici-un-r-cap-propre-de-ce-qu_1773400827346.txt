2) Ce que la nouvelle couche interconnect apporte
-------------------------------------------------
A. Activity Stream
- Timeline unifiée de tous les événements importants:
  - SUGUVAL / SUGUMAILLANE: factures, écritures bancaires, frais généraux, paie, rapports, imports relevés.
  - Foot / Paris: création de pronos, paris placés, résultats, grosses variations de bankroll.
  - Assistant / Système: emails importants, rapports générés, diagnostics, health checks.
- Requêtes possibles:
  - 'Que s’est-il passé le 15/02 ?' -> vue croisée par domaine.
  - 'Montre les événements importants sur Suguval cette semaine' -> filtré par domaine + importance.
  - Vue jour/semaine cross-domain.

B. Entity Links
- Graphe de relations explicite entre entités:
  - Facture <-> écriture bancaire <-> fichier PDF <-> anomalie.
  - Pari <-> match (fixture) <-> prédiction modèle <-> mouvements bankroll.
  - Journée HubRise <-> encaissements banque <-> rapport mensuel.
- Traversal jusqu’à 5 niveaux pour remonter tout le contexte autour d’un point.

C. Entity Tags
- Tagging sémantique commun à tous les domaines:
  - Fournisseurs, ligues, bookmakers, périodes, anomalies, types de paris, etc.
- Permet des requêtes du type:
  - 'Tout ce qui touche #Metro sur 3 mois'
  - 'Tous les #value_bet et leurs résultats'

D. Cross-Domain Intelligence
- Endpoint de 'full entity context':
  - Pour n’importe quelle entité (facture, match, pari, jour, rapport, etc.): activités associées, liens, tags, graphe local.
- Stats globales:
  - Niveau d’interconnexion par domaine.
  - Couverture et zones sous-exploitées (bons candidats d’enrichissement).
