3.1. Types et props
export type AvatarState =
  | "idle"
  | "thinking"
  | "searching"
  | "analyzing"
  | "speaking"
  | "listening"
  | "alert";

export type ConfidenceLevel = "normal" | "verified" | "weak";

interface UlysseAvatarProps {
  isActive?: boolean;
  isSpeaking?: boolean;
  isListening?: boolean;
  isSearching?: boolean;
  isAnalyzing?: boolean;
  orbColor?: string;
  orbIntensity?: number;
  isPaused?: boolean;
  reducedMotion?: boolean;
  className?: string;

  // NOUVEAU
  confidence?: ConfidenceLevel;   // "verified" = données béton, "weak" = prudence
  hasVerifiedData?: boolean;      // alias rapide pour mettre en avant Verified Scraper
  isTranslating?: boolean;        // Translation Core actif
}
3.2. Utilisation de confidence pour surcouche visuelle
Juste avant le return, après const c = STATE_COLORS[state];:

const confidence = (confidenceLevel: ConfidenceLevel | undefined): ConfidenceLevel =>
  confidenceLevel || "normal";

const cnf = confidence(confidenceLevel);
Et dans le JSX, près du core (autour des circles de centre 170,165), tu peux ajouter un overlay conditionnel :

{/* Overlay confiance VERIFIED */}
{(cnf === "verified" || hasVerifiedData) && (
  <motion.circle
    cx="170"
    cy="165"
    r="60"
    fill="none"
    stroke="#22c55e"            // vert confiance
    strokeWidth="3"
    strokeOpacity="0.8"
    filter="url(#glow-md)"
    animate={isPaused ? {} : { r: [58, 64, 58], opacity: [0.5, 0.9, 0.5] }}
    transition={{ duration: 1.8, repeat: Infinity, ease: "easeInOut" }}
  />
)}

{/* Overlay confiance WEAK */}
{cnf === "weak" && (
  <motion.circle
    cx="170"
    cy="165"
    r="60"
    fill="none"
    stroke="#f97316"            // orange avertissement
    strokeWidth="2"
    strokeOpacity="0.8"
    strokeDasharray="6 6"
    filter="url(#glow-sm)"
    animate={isPaused ? {} : { opacity: [0.3, 0.8, 0.3] }}
    transition={{ duration: 1.5, repeat: Infinity, ease: "easeInOut" }}
  />
)}
3.3. Indicateur Translation Core
Dans le même groupe central, tu peux ajouter un arc interne :

{/* Arc Translation Core */}
{isTranslating && (
  <motion.path
    d={`
      M 130 165
      A 40 40 0 0 1 210 165
    `}
    fill="none"
    stroke="#06b6d4"          // cyan
    strokeWidth="4"
    strokeLinecap="round"
    filter="url(#glow-sm)"
    animate={isPaused ? {} : { strokeDashoffset: [0, 40, 0] }}
    transition={{ duration: 1.2, repeat: Infinity, ease: "easeInOut" }}
  />
)}
Ça dessine un arc en bas (ou haut selon la coordonnée) qui pulse quand Translation Core est actif.
