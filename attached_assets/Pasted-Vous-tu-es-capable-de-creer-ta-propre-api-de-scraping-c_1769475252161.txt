2️⃣ Stack technique sans API externe
Tout en local / serveur à toi :

Node.js + TypeScript (aligné avec ton stack)
Crawlee pour la base crawler/scraper
HTTP simple (Cheerio) + Browser (Playwright/Puppeteer) gérés
Queues, politeness, retry, gestion d’erreurs
Cheerio pour parser HTML côté serveur (fast, léger)
Playwright pour les sites JS lourds (aucun appel à un service externe)
Express / Fastify pour exposer l’API
Aucune API externe type ScrapingBee, Zyte, etc.
Tu contrôles tout, chez toi.

3️⃣ Architecture propre
3.1. Couches
scraperService.ts
Fonction générique crawlAndScrape(options)
Logique Crawlee + Cheerio + Playwright
extractors.ts
Helpers pour extraire du texte, des liens, appliquer des sélecteurs CSS
router.ts
Expose /api/scraper/crawl et éventuellement:
/api/scraper/sitemap
/api/scraper/custom-profile (pour profils prédéfinis genre “site de foot X”)
3.2. Modes
mode: "http" → CheerioCrawler (rapide, HTML pur)
mode: "browser" → PlaywrightCrawler (site JS, logins, etc.)
mode: "auto" → tentative HTTP, fallback navigateur si besoin (ex: pas de contenu, trop peu de texte, etc.)
4️⃣ Exemple de squelette en TypeScript (Crawlee)
Je te mets un bout concret pour que tu vois le niveau de détail que je peux produire pour toi :

// scraperService.ts
import { CheerioCrawler, PlaywrightCrawler } from 'crawlee';
import * as cheerio from 'cheerio';

export type ScrapeMode = 'auto' | 'http' | 'browser';

export interface ScrapeRequest {
  url: string;
  mode?: ScrapeMode;
  depth?: number;
  maxPages?: number;
  extract?: {
    text?: boolean;
    html?: boolean;
    links?: boolean;
    metadata?: boolean;
    selectors?: Record<string, string>; // nom -> sélecteur CSS
  };
}

export interface ScrapedPage {
  url: string;
  status: number;
  metadata?: {
    title?: string;
    contentType?: string;
  };
  links?: string[];
  text?: string;
  html?: string;
  extracted?: Record<string, any>;
}

export interface ScrapeResult {
  pages: ScrapedPage[];
}

export async function crawlAndScrape(req: ScrapeRequest): Promise<ScrapeResult> {
  const pages: ScrapedPage[] = [];
  const visited = new Set<string>();

  const {
    url,
    mode = 'auto',
    depth = 0,
    maxPages = 50,
    extract = { text: true, links: true, metadata: true }
  } = req;

  const shouldFollowLinks = depth > 0;

  const handleCheerioPage = async ({ request, response, body, $ }: any) => {
    if (pages.length >= maxPages) return;

    const currentUrl = request.loadedUrl as string;
    if (visited.has(currentUrl)) return;
    visited.add(currentUrl);

    const contentType = response?.headers?.['content-type'] ?? '';
    const page: ScrapedPage = {
      url: currentUrl,
      status: response?.statusCode ?? 0,
      metadata: {}
    };

    if (extract.metadata) {
      page.metadata = {
        title: $('title').first().text().trim(),
        contentType
      };
    }

    if (extract.text) {
      page.text = $('body').text().replace(/\s+/g, ' ').trim();
    }

    if (extract.html) {
      page.html = body?.toString() ?? '';
    }

    if (extract.links) {
      const links: string[] = [];
      $('a[href]').each((_, el) => {
        const href = $(el).attr('href');
        if (href && !href.startsWith('#')) {
          links.push(new URL(href, currentUrl).toString());
        }
      });
      page.links = [...new Set(links)];
    }

    if (extract.selectors) {
      const result: Record<string, any> = {};
      for (const [key, selector] of Object.entries(extract.selectors)) {
        const elements = $(selector);
        if (elements.length === 0) {
          result[key] = null;
        } else if (elements.length === 1) {
          result[key] = elements.first().text().trim();
        } else {
          result[key] = elements
            .map((_, el) => $(el).text().trim())
            .get()
            .filter(Boolean);
        }
      }
      page.extracted = result;
    }

    pages.push(page);

    if (shouldFollowLinks && page.links) {
      for (const link of page.links) {
        if (!visited.has(link) && pages.length < maxPages) {
          await crawler.addRequests([{ url: link }]);
        }
      }
    }
  };

  // HTTP crawler par défaut
  const crawler = new CheerioCrawler({
    maxRequestsPerCrawl: maxPages,
    maxConcurrency: 5,
    requestHandlerTimeoutSecs: 60,
    async requestHandler(ctx) {
      await handleCheerioPage(ctx);
    },
    failedRequestHandler({ request }) {
      console.warn(`Request failed: ${request.url}`);
    }
  });

  await crawler.run([url]);

  // TODO: si mode === 'auto' et que la page a l’air “vide / JS-only” → fallback PlaywrightCrawler

  return { pages };
}
Et côté route Express :

// scraperRouter.ts
import express from 'express';
import { crawlAndScrape } from './scraperService';

const router = express.Router();

router.post('/crawl', async (req, res) => {
  try {
    const result = await crawlAndScrape(req.body);
    res.json(result);
  } catch (err: any) {
    console.error('Scraper error:', err);
    res.status(500).json({ error: 'Scrape failed', details: err.message });
  }
});

export default router;

5️⃣ Gestion des limites / respect / robustesse
Sans API externe, il faut être carré :

Respect des sites
lire/optionnellement respecter robots.txt
user-agent propre (ex: UlysseScraperBot/1.0)
délais entre requêtes, limite de profondeur
Anti-ban light
random delay, headers “human-like”
option proxy (si un jour tu veux)
Résilience
retry sur 429/5xx
timeout par requête
log + stockage des pages en erreur pour debug
