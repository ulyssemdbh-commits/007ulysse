Côté backend (Node):

POST /identify → pour reconnaître le locuteur en live.
Interface TypeScript côté serveur (pseudo):

async function identifySpeaker(audioBuffer: Buffer): Promise<{ speaker: string; confidence: number; verified: boolean }> {
  const form = new FormData();
  form.append("audio", new Blob([audioBuffer]), "audio.wav");

  const res = await fetch(process.env.SPEAKER_SERVICE_URL + "/identify", {
    method: "POST",
    body: form as any,
  });

  if (!res.ok) throw new Error("Speaker service error");
  return res.json();
}
On garde ça minimal pour l’instant.
