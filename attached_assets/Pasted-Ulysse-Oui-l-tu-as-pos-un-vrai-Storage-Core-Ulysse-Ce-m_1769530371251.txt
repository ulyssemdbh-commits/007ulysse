1️⃣ Centraliser la gestion owner / approved (sécurité logique)
Tu as approvedUsers mais les méthodes de storage ne savent pas si l’appelant est:

l’owner (toi),
un approved (tes filles),
un externe (Alfred).
Pour renforcer la logique “Ulysse n’est que pour toi + approuvés” tu peux ajouter une petite couche au-dessus du storage:

soit un AccessControlService,
soit des helpers dans le Storage pour savoir rapidement:
async isOwner(userId: number): Promise<boolean> {
  const user = await this.getUser(userId);
  return !!user?.isOwner;
}

async isOwnerOrApproved(userId: number): Promise<boolean> {
  const user = await this.getUser(userId);
  if (!user) return false;
  if (user.isOwner) return true;
  const approved = await this.isUserApproved(userId);
  return approved;
}
Et ensuite, dans ton API / routes:

Tu peux refuser certaines opérations si !isOwnerOrApproved(userId)
(par ex. accès à certains homeworks, settings, SUGU plus tard, etc.).

async addApprovedUser(data: InsertApprovedUser): Promise<ApprovedUser> {
  const count = await this.getApprovedUserCount();
  if (count >= MAX_APPROVED_USERS) {
    throw new Error(`Maximum of ${MAX_APPROVED_USERS} approved users allowed`);

  const existing = await db.select().from(approvedUsers).where(eq(approvedUsers.userId, data.userId));

3️⃣ Stocker le UlysseCharter par défaut en DB (au lieu de juste le retourner)
Actuellement, getCharter:

renvoie un charter par défaut si rien en DB,
mais ne le persiste pas.
Tu peux faire en sorte que le premier getCharter crée automatiquement une entrée:

async getCharter(userId: number): Promise<UlysseCharter> {
  const [existing] = await db
    .select()
    .from(ulysseCharter)
    .where(eq(ulysseCharter.userId, userId));

  if (existing) return existing;

  const now = new Date();
  const defaults: InsertUlysseCharter = {
    userId,
    communicationStyle: "direct",
    language: "fr",
    responseLength: "concise",
    priorityDomains: [],
    activeProjects: [],
    behaviorRules: [],
    wakeWord: "Ulysse",
    voicePersonality: "professional",
    rememberConversations: true,
    contextRetentionDays: 30,
    proactiveInsights: true,
    dailyBriefEnabled: true,
    dailyBriefTime: "08:00",
    customInstructions: null,
    createdAt: now,
    updatedAt: now,
  };

  const [created] = await db.insert(ulysseCharter).values(defaults).returning();
  return created;
}
→ Ça te garantit que tout ce qui touche à moi (charte) est réellement dans la DB, pas juste en mémoire.
